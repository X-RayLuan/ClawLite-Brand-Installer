# Thread Patterns

스레드 유형별 패턴과 훅(Hook) 예시 모음.

## Hook Patterns (첫 트윗)

### 1. 충격 통계형
```
[놀라운 숫자]가 [의외의 사실]이라는 걸 아시나요?

대부분의 사람들이 모르는 이야기 🧵
```

### 2. 질문형
```
왜 [현상]이 일어날까요?

[N]가지 이유를 정리했습니다 👇
```

### 3. 대담한 주장형
```
[통념]은 완전히 틀렸습니다.

[기간] 동안 [경험]을 하며 깨달은 것들 🧵
```

### 4. 스토리형
```
[시점]에 [상황]이 벌어졌습니다.

그 뒤에 일어난 일이 모든 걸 바꿨습니다 👇
```

### 5. 리스트 예고형
```
[주제]에 대해 알아야 할 [N]가지.

(특히 [N]번째가 중요합니다) 🧵
```

## Thread Types

### 교육형 (How-to)
- Hook: 문제 제시 → "해결법을 찾았음"
- Body: 단계별로 1 트윗 = 1 스텝
- CTA: "도움이 됐다면 RT로 공유해주세요"

### 분석형 (Breakdown)
- Hook: 대상 소개 + 왜 분석할 가치가 있는지
- Body: 핵심 포인트를 숫자/데이터와 함께 전달
- CTA: "더 깊은 분석이 필요하면 팔로우"

### 큐레이션형 (List)
- Hook: "N개를 엄선했습니다"
- Body: 각 항목을 한 트윗에 소개
- CTA: "북마크하고 나중에 다시 보세요 🔖"

### 스토리형 (Narrative)
- Hook: 긴장감 있는 시작
- Body: 시간순 또는 인과 흐름
- CTA: 교훈 + 공유 요청

### 의견형 (Opinion)
- Hook: 논쟁적인 주장
- Body: 근거를 하나씩 제시
- CTA: "어떻게 생각하시나요? 의견을 남겨주세요"

## CTA Patterns (마지막 트윗)

### 공유 유도
```
이 스레드가 도움이 됐다면
첫 트윗을 RT 해주세요.

더 많은 사람에게 도달할 수 있습니다 🙏
```

### 팔로우 유도
```
이런 [주제] 콘텐츠를 매주 올립니다.

놓치지 않으려면 팔로우 🔔
```

### 토론 유도
```
여러분의 생각은 어떤가요?

댓글로 의견을 남겨주세요 💬
```

### 저장 유도
```
나중에 다시 볼 수 있도록
이 스레드를 북마크 해두세요 🔖
```

## Length Guidelines

| 콘텐츠 유형 | 권장 트윗 수 |
|---|---|
| 짧은 팁/인사이트 | 3~5 |
| 기사/블로그 요약 | 5~8 |
| 심층 분석/가이드 | 8~12 |
| 스토리/케이스 스터디 | 10~15 |
